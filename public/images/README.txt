EV Bike Images Instructions

1. Download high-quality EV bike images from a stock photo site or your own collection.
2. Rename the main image file to "ev-bike.jpg" or "ev-bike.png".
3. Place the image files in this directory (public/images/).
4. Images should be optimized for web (compressed to a reasonable file size).
5. Recommended resolution: at least 1200x800 pixels for good quality.

Suggested sources for royalty-free stock images:
- Pexels (https://www.pexels.com/search/electric%20bike/)
- Pixabay (https://pixabay.com/images/search/electric%20bike/)
- Unsplash (https://unsplash.com/s/photos/electric-bike)

After adding the images, restart the development server with:
npm run dev
